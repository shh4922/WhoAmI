# WhoAmI
## 📝 시작동기

교내 프로젝트로 딥러닝을 이용한 서비스를 만들라고하셨다.
그래서 게임캐릭터의 일러스터 이미지를 학습시켜 사람의 얼굴을 사진촬영시, 닮은 게임캐릭터가 보여지는 앱 애플리케이션을 만들면 재미있겠다..
생각해서 시작하게됌

## ✋ 역할

개인프로젝트


## ⚒️ 사용기술

`kotlin`, `flask`, `retrofit2`
